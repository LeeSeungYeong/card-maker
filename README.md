# Business Card Maker
- react-router로 로그인화면과 main 화면 경로 설정
- cloudinary를 사용하여 이미지 파일 편집후 업로드
- firebase를 사용하여 구글과 깃허브 로그인을 구현하였고, 로그인한 ID별로 데이터를 저장
